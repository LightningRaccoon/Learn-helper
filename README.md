# Python Learn Helper

This project was made to study for Adatb√°zisok II EA exam using a flip card interface and a quick quiz section. It is a simple project with simple functionalities. Note that, it is still under development.

## Features

1. **Flip Card Interface**
   - View a question prompt on one side of the card.
   - Flip the card to reveal the answer.

2. **Quiz Section**
   - Take a quiz consisting of 4 multiple-choice questions.
   - Get instant feedback on your answers.
